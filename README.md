# Brand Yourself, Earn More — Non-Tech Edition

CareerByteCode landing site for non-tech professionals (Freelancing Program + Non-Tech Creator Track).
Static HTML/CSS/JS — no build step. Accent theme: violet/purple.

## Pages
- index.html — landing (non-tech: HR, teachers, coaches, marketers, returnees, parents)
- syllabus.html — full curriculum (9 phases, 11 service modules, 65+ skills)
- earn.html — Grow & Earn playbook (60 visibility plays, digital-marketing services, 7-level ladder)

## Deploy on GitHub Pages
1. Create repo named freelancing-nontechcourse and upload all files (keep fonts/ as a subfolder, .nojekyll at root).
2. Settings → Pages → Build and deployment → Source → Deploy from a branch → main → / (root) → Save.
3. Live in ~1 min at https://careerbytecode.github.io/freelancing-nontechcourse/
